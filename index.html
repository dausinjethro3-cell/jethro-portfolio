About Me

Hi! I’m Jethro, a highly motivated aspiring Software Developer passionate about turning ideas into interactive digital experiences.

I started my web development journey with HTML and CSS, building a working portfolio layout within just two days of learning. From there, I quickly expanded my skills into JavaScript, creating multiple projects like calculators, counters, and simple games within hours of study.

I also have a strong foundation in Python, where I’ve built practical systems such as a receipt generator and calculator. My learning style focuses on understanding the logic behind the code, allowing me to debug efficiently and explore how to maximize each language’s potential.

Beyond coding, I enjoy designing user interfaces through Figma and Penpot, combining creativity and functionality to produce clean, user-friendly designs.

I’m continuously improving my skills toward becoming a Software Developer, with an interest in backend technologies like Python and databases such as MySQL.

I’m driven by curiosity, persistence, and a genuine excitement for creating something new from concept to code. This project was built during my learning of HTML and CSS.
